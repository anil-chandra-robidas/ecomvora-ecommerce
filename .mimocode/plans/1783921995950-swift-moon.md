# Plan: Redesign Site Layout Inspired by PikMart Reference

## Reference Design Analysis (PikMart)
The reference shows a modern e-commerce layout with:
1. **Top announcement bar** — promotional text with countdown timer
2. **Two-row navbar** — Row 1: Logo, search bar, location/wishlist/cart/login icons. Row 2: "Browse Categories" dropdown + nav links (Shop, Super Deals, Find Store, What's New, Original Offer, Pages)
3. **Hero grid** — Asymmetric grid of promotional banners (large left + large right + 2 medium below)
4. **Weekly Best Deals** — Category filter tabs + product grid with countdown timers, ratings, delivery info
5. **Bottom promo grid** — 3 promotional cards with countdown timers

## Design Approach
Keep our **dark theme** (#0a0a0a base, orange #f97316 brand) but match the **layout structure** from the reference. Adapt food/grocery content to the PikMart grid layout.
