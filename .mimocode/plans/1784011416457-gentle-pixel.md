## Implementation Details

### API Routes (`src/app/api/admin/blog/`)

**GET `/api/admin/blog`** — List all blog posts
- Query params: `?published=true` (filter published only), `?search=...` (search title/excerpt)
- Returns: JSON array of BlogPost records, ordered by `createdAt desc`

**POST `/api/admin/blog`** — Create blog post
- Body: `{ title, slug, excerpt, content, image, category, author, readTime, published }`
- Returns: created BlogPost with status 201

**GET `/api/admin/blog/[id]`** — Get single blog post by ID

**PUT `/api/admin/blog/[id]`** — Update blog post
- Body: same fields as POST

**DELETE `/api/admin/blog/[id]`** — Delete blog post

Follow the exact patterns from `src/app/api/products/route.ts` and `src/app/api/products/[id]/route.ts` (NextRequest/NextResponse, `params: Promise<{ id: string }>`, prisma singleton).

### Admin Blog List Page (`src/app/admin/blog/page.tsx`)

Follow the product list page pattern:
- Table columns: Title (with image thumbnail), Category, Author, Status (Published/Draft badge), Date, Actions (Edit/Delete)
- Search bar filtering by title/category
- "Add Blog Post" button linking to `/admin/blog/new`
- Delete with `window.confirm()` + optimistic state removal

### Admin Blog Create/Edit Pages

Form fields:
- **Title** (text input)
- **Slug** (text input, auto-generated from title using `slugify` from `@/lib/utils`)
- **Category** (select: Nutrition, Seasonal, Lifestyle, Tips, Education, Recipes)
- **Author** (text input)
- **Read Time** (text input, e.g. "5 min read")
- **Excerpt** (textarea, 2-3 lines)
- **Content** (large textarea, supports HTML)
- **Image URL** (text input with preview)
- **Published** (checkbox toggle)

### Frontend Blog Pages

**Blog listing (`src/app/blog/page.tsx`)**:
- Replace hardcoded `posts` array with `fetch("/api/admin/blog?published=true")`
- Use `createdAt` from DB for date display (format as "Jan 15, 2025")

**Blog single (`src/app/blog/[slug]/page.tsx`)**:
- Fetch by slug from API (new endpoint or query param)
- Use `dangerouslySetInnerHTML` for content (same as now)

**Homepage Blog component (`src/components/home/Blog.tsx`)**:
- Fetch latest 3 published posts from API
